# Slot_MachineV1.pde
吃角子老虎機--processing

2013/04 小專題，與兩名組員合力作業，首次嘗試遊戲製作(大二)，為基本拉霸機，加上互動式投幣動畫及互動拉桿，及"7"通過中間時發出音效。
<ul>
<li>使用Processing (JAVA)開發</li>
<li>美術使用Illustrator開發</li>
<li>配合音樂律動的上面板</li>
<li>按777combo提高速度及獎勵</li>
</ul>
